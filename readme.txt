Create sidebar widgets to display a list of your custom taxonomy terms with ease!

== Description ==

Create sidebar widgets to display a list of your custom taxonomy terms with ease!  You have total control over many options from specifying the maximum number of terms to be displayed all the way to specifying how you want your terms to be ordered.  You can even display terms from multiple custom taxonomies in a single widget, woah!  Go ahead, give it a whirl and tell me what you think!

== Installation ==

Installation is as easy as 1, 2, 3!  Simply download the plugin and upload it into your WordPress installations plugins directory, go to your site's plugins page within the admin panel, activate "Custom Taxonomy Sidebar Widget," and sit back and relax!
